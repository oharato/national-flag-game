# 画面設計書（ワイヤーフレーム）

本ドキュメントは、国旗学習アプリの各画面のレイアウトと要素を定義します。

## レスポンシブデザイン方針

### モバイルファースト設計
- スマートフォンでの表示を最優先に設計
- Tailwind CSSのブレークポイント（`md:`プレフィックス）を使用してPC版を拡張

### 主な適応的レイアウト
- **フォームコンポーネント**:
  - スマホ: 小さいパディング（`px-2 py-1.5`）、フォントサイズ（`text-sm`）
  - PC: 標準サイズ（`px-3 py-2`）、フォントサイズ（`text-base`）
  
- **ラベル表示**:
  - ランキング画面: スマホではラベル非表示でコンパクト化
  - クイズ設定画面: 全画面サイズでラベル表示

- **ランキング表示**:
  - PC: テーブル形式（順位・ニックネーム・スコア・日時を横並び）
  - スマホ: カード形式（順位・ニックネームを左、スコアを右に配置、日時は下部）

- **エラー表示**:
  - alertダイアログを廃止し、画面内にコンパクトに表示
  - 入力欄の枠線色変更とエラーメッセージで視覚的フィードバック

## 1. トップページ

シンプルなエントリーポイント。ユーザーをクイズモードか学習モードに誘導します。

```
+------------------------------------------------------+
|                                                      |
|                国旗学習アプリ (仮)                   |
|                                                      |
+------------------------------------------------------+
|                                                      |
|                                                      |
|           +--------------------------+               |
|           |      クイズに挑戦する    |               |
|           +--------------------------+               |
|                                                      |
|           +--------------------------+               |
|           |      国旗を学習する      |               |
|           +--------------------------+               |
|                                                      |
|                                                      |
+------------------------------------------------------+
```

---

## 2. クイズ設定画面

クイズを開始する前の設定を行います。

```
+------------------------------------------------------+
| < トップページに戻る                                 |
+------------------------------------------------------+
|                                                      |
| ニックネームを入力:                                  |
| +--------------------------------------------------+ |
| | Guest123                                         | |
| +--------------------------------------------------+ |
|                                                      |
| クイズ形式:                                          |
| (o) 国旗を見て国名を選ぶ                             |
| ( ) 国名を見て国旗を選ぶ                             |
|                                                      |
| 出題範囲:                                            |
| [ 全世界 v ]  (ドロップダウン)                       |
|                                                      |
|           +--------------------------+               |
|           |          スタート        |               |
|           +--------------------------+               |
|                                                      |
+------------------------------------------------------+
```

```
| |                                                  | |
| |                [国旗の画像]                      | |

| |           (クリックで詳細表示)                   | |

| +--------------------------------------------------+ |

|      [< 前へ]         [次へ >]                      |
|                                                      |
+------------------------------------------------------+
```
